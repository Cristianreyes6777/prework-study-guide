CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Much how HTML was the framing, CSS is the color and style. Programmers use CSS to define the aesthetics of the webpage
  • There are three ways to style CSS sheets: Inline, Internal, External.
  • Inline is used when trying to style the HTML file directly
  • Internal is for styling in b/w elements.
  • External is the most perferred and is the method we used om the prework study guide.
